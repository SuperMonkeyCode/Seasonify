
# Installing :
```
$ yarn install
$ react-native link native-base
```

# Launching (dev) :
* `$ npm start`
* Launch the app ( or `$ npm run android` if not installed)
* Dev menu (ctrl+M or ctrl+F2) -> Start Remote JS Debugging
